      # 开始使用

      # 运行项目

      # clone
git clone git@github.com:JeremyWu917/jeremy-press.git

# change dir
cd jeremy-press

# install dependencies
yarn

# run dev
yarn dev

      TIP

      现在您可以在 http://localhost:8080/jeremy-press/ 下访问网站的主页了 ☕️
